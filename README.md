# Team Task Manager

A task management project for organizing team work, assigning responsibilities, and tracking progress across statuses.

## Planned Features

- Create, edit, and delete tasks
- Assign tasks to team members
- Track task status such as To Do, In Progress, and Done
- Set priorities and due dates
- Filter tasks by status, owner, or priority
- Dashboard view for team progress

## Suggested Tech Stack

- Frontend: HTML, CSS, JavaScript or React
- Backend: Node.js and Express
- Database: MongoDB or PostgreSQL
